# Stick Hero with Canvas

A fun, addictive stick hero game built with HTML5 Canvas, JavaScript, and CSS.

## Description

Stick Hero with Canvas is a browser-based implementation of the popular Stick Hero game. The player must stretch a stick to bridge the gap between platforms, aiming to reach the next platform without falling. Score points by successfully crossing and collect bonuses for perfect landings.

## Features

- Intuitive mouse controls: Hold to stretch the stick, release to drop it.
- Dynamic scoring system with double points for perfect bridges.
- Responsive canvas-based graphics.
- Restart functionality to play again instantly.

## Gameplay

1. Click and hold the mouse to stretch the stick.
2. Release to drop the stick and see if it reaches the next platform.
3. If successful, the stick hero walks across; if not, game over.
4. Score increases with each successful bridge.
5. Perfect landings (stick exactly matches the platform) award double points.

1. Clone the repository:

2. Open `index.html` in your web browser.
3. Start playing!

Alternatively, visit the live demo: [https://binaryvortex.github.io/Stick-Hero-with-Canvas/](https://binaryvortex.github.io/Stick-Hero-with-Canvas/)
